# garden
a garden-simulator using cellular automata, written using pygame.

## To Run:

`python ./src/main.py`

`S` - sand
`W` - water
`R` - rock
`F` - fire
`C` - reset
`B` - clear


## Features:

- sand
- water
- rocks
- fire
