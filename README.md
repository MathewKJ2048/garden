# garden
a garden-simulator using cellular automata, written using pygame.

## To Run:

`python ./src/main.py`

`S` - sand
`W` - water
`R` - rock
`F` - fire
`A` - acid
`C` - reset
`I` - inert material
`B` - clear
`G` - flip gravity
`L` - stats


## Features:

- sand
- water
- rocks
- fire
