# garden
a garden-simulator using cellular automata, written using pygame.

## To Run:

`python ./src/main.py`

`S` - sand  
`I` - ice  
`W` - water  
`R` - rock  
`F` - fire  
`A` - acid  
`O` - oil  
`E` - embers  
`V` - inert material  
`C` - nothing  
`B` - erase  
`L` - lava  
`M` - seeds

`P` - pause/play
`G` - flip gravity  
`/` - stats  

Seeds must be added on top of sand and watered to grow
