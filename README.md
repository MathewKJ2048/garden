# garden
a garden-simulator using cellular automata, written using pygame.

## To Run:

`python ./src/main.py`

`S` - sand  
`W` - water  
`R` - rock  
`F` - fire  
`A` - acid  
`O` - oil
`E` - embers
`I` - inert material  
`C` - nothing  
`B` - erase  
`G` - flip gravity  
`L` - stats
